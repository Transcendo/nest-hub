# business-idea 项目目的

## 项目定位

business-idea 是一个长期公开研究项目，但首先服务于项目维护者自己。

### 核心目标
1. 每日获取一个值得研究的创业 idea / 创业公司 / 创业方向。
2. 以开源方式公开沉淀，方便其他相关用户查阅和了解。
3. 所有内容必须尽量基于真实、可信、可追溯的公开来源。
4. 文风必须统一，不能像临时拼接的日报，也不能像营销软文。

## 内容要求

- 优先使用可信公开来源，例如：
  - Hacker News
  - 公司官网
  - 官方博客
  - Wikipedia
  - 官方文档
  - 主流可信媒体 / 数据库
- 不接受明显营销稿、软文、二手拼贴垃圾内容作为主来源。
- 内容应兼顾：
  - 自己用于判断创业 idea
  - 对外部读者可读、可查、可复用

## 后续工作方向

1. 建立统一文风规范。
2. 评审并改造外部可参考 skill（如 KKKKhazix/Khazix-Skills）为自己的版本。
3. 用已有 daily pick 内容重写一批页面，测试最终文风和结构。
