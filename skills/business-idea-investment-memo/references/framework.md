# Framework

这个项目推荐吸收三类框架，而不是迷信单一模板。

## 1. 创业教练视角
核心问题：
- 这家公司在哪个赛道
- 它为什么值得做
- 它的难点是什么
- 如果我是创业者，最应该先解决什么

## 2. VC / FA 视角
核心问题：
- 市场够不够大
- 产品有没有差异化
- 商业化是不是成立
- 团队是否撑得住
- 风险在哪里

## 3. 商业模式视角
核心问题：
- 客户是谁
- 价值主张是什么
- 怎么收费
- 核心资源和关键活动是什么
- 为什么别人不容易复制

## 推荐固定检查项
- 赛道
- 产品定义
- 用户需求
- 优点
- 缺点 / 风险
- 商业路径
- 竞争格局
- 最终判断
