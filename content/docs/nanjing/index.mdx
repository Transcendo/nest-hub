# 南京租房指南

南京租房先看办公区和通勤带，再看公司名。雨花台 / 软件谷、江北新区、江宁滨江、河西 / 南站、徐庄 / 仙林分别对应不同的房源形态、地铁接驳、过江成本和晚归风险。刚拿到 offer 时，先问清实际打卡楼栋、园区入口、班车、驻场和常见下班时间，再决定住近场、换乘带，还是短租过渡。

## 怎么使用这个总览

1. 先在总览里判断 offer 属于哪条通勤带：雨花台 / 软件谷、江北新区、江宁滨江、河西 / 南站，还是徐庄 / 仙林。
2. 再进入对应公司或片区指南，看候选居住区、通勤路径、预算取舍和看房风险。
3. 拿到 offer 后优先问清实际打卡楼栋、园区入口、是否驻场、是否有班车、晚归频率。
4. 看房当天用正规平台、线下实看、通勤实测和合同条款交叉确认，不只看“离公司近”。
5. 如果还没入职，先用 1-3 周过渡房把真实通勤跑出来，再决定是否签一年。

## 地图判断原则

南京不要只按“市中心 / 公司附近”做租房判断：

- **雨花台 / 软件谷南侧**：重点看软件大道、宁双路、凤信路、安德门、小行、铁心桥、西善桥、油坊桥和南京南站外扩。
- **江北新区 / 高新区**：先决定是否每天过江，再比较江北近场、浦口地铁沿线和主城换乘方案。
- **江宁滨江**：班车、园区入口和晚归路线比直线距离更关键，不能直接套江宁主城逻辑。
- **河西 / 南京南站**：适合双通勤、跨城和换乘兜底，但租金、噪音和日常通勤稳定性要一起算。
- **徐庄 / 仙林 / 麒麟**：先核玄武大道、徐庄软件园或仙林方向的实际楼栋，再比较近场、马群、孝陵卫和主城换乘。

## 预算和通勤先后顺序

| 状态 | 优先策略 | 不建议 |
| --- | --- | --- |
| 楼栋明确、每天到岗 | 先把近场、地铁 / 班车沿线和晚归路线跑通，再比较房型 | 为了多省几百元，把通勤拉到每天不稳定 |
| 楼栋明确但晚归少 | 可以看换乘带或外扩区域，用面积 / 预算换通勤 | 只看地铁站名，不算园区门岗和步行 |
| 楼栋未确认、试用期或驻场 | 短租、月租公寓、可转租合租或换乘兜底 | 押二付三、长违约、一年不可转租 |
| 情侣 / 室友双通勤 | 把两个人办公点和晚归时间放进同一张表 | 只满足一个人近场，让另一方每天过江 / 跨城 |
| 预算敏感 | 先压押付和总成本，再看合租、老小区和公寓 | 只比挂牌月租，漏算服务费、中介费、商水商电和通勤费 |

## 南京找房方式

- **公司附近先看样本，不急着签长租**：南京很多办公点不在传统市中心，第一周先用短租、公寓或同事样本校准路线。
- **地铁只是第一层**：软件谷、江北、江宁滨江和徐庄都要测楼栋到地铁口 / 园区门的最后一公里。
- **晚归路线必须实测**：晚上 21:30 后再看公交、共享单车、打车点、小区门禁和路灯。
- **费用拆清楚**：中介费、服务费、押付、民水民电、转租规则、退租违约金都要写进合同或聊天记录。
- **平台只当线索池**：链家 / 贝壳、品牌公寓、门店和同事推荐都要回到线下看房、合同主体和实时可租状态。

## 南京租房 FAQ

### 刚拿到南京科技公司 offer，第一套房先看哪里？

先看真实办公点。雨花台 / 软件谷、江北新区、江宁滨江、徐庄 / 仙林和河西 / 南站的通勤逻辑完全不同。楼栋没确认时，先看南京大厂 Offer 核验页和城市总览，不要直接按“公司名 + 附近租房”下定。

### 江北新区上班，要不要住主城？

每天江北打卡的人，不建议默认住主城。主城过江适合伴侣 / 室友在主城、每周去江北少、或确实需要河西 / 鼓楼生活半径的人；否则过江高峰、晚归打车和通勤疲劳会很快吞掉“住得更中心”的好处。

### 软件谷和南京南站怎么取舍？

软件谷近场更适合高频到岗、晚归多和通勤风险低的人；南京南站更适合双通勤、跨城、楼栋未确认或需要换乘兜底的人。签约前必须实测房门到楼栋，不要只看到地铁站的时间。

### 实习或试用期适合直接签一年吗？

不适合。除非楼栋、到岗频率、团队稳定性、合同退租和转租条款都已经确认，否则先用短租、月租公寓或可转租合租过渡，降低押金和二次搬家的损失。

### 南京租房最容易漏算什么？

最容易漏算园区最后一公里、过江成本、晚归打车、服务费、中介费、商水商电、提前退租违约和转租手续费。比较房源时要算总成本，不要只看月租。
