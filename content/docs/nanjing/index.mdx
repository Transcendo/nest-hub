# 南京租房指南

南京租房不能只看“离市中心远不远”。雨花台 / 软件谷南侧、江北新区、江宁滨江产业基地、河西 / 南站换乘带、徐庄 / 仙林扩展带，对应的是完全不同的通勤、房源和晚归风险。

<NanjingOverview />

## 当前使用方式

1. 先在总览里判断你的 offer 属于雨花台 / 软件谷、江北新区、江宁滨江、河西/南站换乘，还是徐庄/仙林扩展带。
2. 拿到 offer 后先向 HR 或同事确认实际打卡楼栋、园区门、班车、常见下班时间，不要先被低价房源牵着走。
3. 把候选房源分成近场、换乘、外扩三类，再实测门到门通勤，尤其是过江、园区最后一公里和晚归路线。
4. 看房当天用正规平台、线下实看和合同条款交叉核验；本项目不把第三方截图或社交帖当稳定事实。

## 后续核验方向

南京已经从城市总览进入指南拆分阶段，并已覆盖雨花台 / 软件谷南侧、江北、江宁滨江、河西 / 南站和徐庄 / 仙林等核心租房题。后续继续只挂实际存在、且有可信办公锚点或明确片区决策价值的页面；没有稳定楼栋的大厂团队先走 offer 核验流程：

| --- | --- | --- |

## 资料来源与口径

南京页优先使用公司官网、投资者关系页、政府/园区公开信息。页面中的区域建议是 NestHub 面向年轻租房者的编辑判断，不代表公司或园区官方建议，也不构成房源推荐。

当前版本只做办公片区、公司样本与通勤方向判断；大厂南京团队先按“候选核验队列”处理，不把招聘城市、工商注册地、旧新闻或驻场项目当作稳定办公地址；不写未经核验的实时租金，不使用来源不明接口、小红书/微信群截图或第三方 SEO 采集页作为事实来源。
