# 爱奇艺成都租房指南

<Callout type="success" title="先看结论">
爱奇艺成都相关岗位先不要按“爱奇艺 = 某一栋楼”直接租房。可信公开信息能确认成都高新区曾把爱奇艺列为新经济头部企业投资项目样本，爱奇艺官方招聘站也提供招聘入口；但具体团队楼栋、是否固定成都办公室、是否客户现场或跨园区协作，必须由 HR / 直属同事确认。没拿到楼栋前，优先用高新南区 / 天府软件园、大源、天府三街、新川和火车南站做候选圈，不要为了低价直接签远端长租。

## 第一步：先确认实际办公楼栋

爱奇艺在成都的公开信息要分两层看：

1. **公司 / 招聘事实**：爱奇艺官方招聘站是岗位与招聘入口，适合你确认岗位归属、招聘类型和后续 HR 联系方式。
2. **成都项目口径**：第一财经关于成都高新区新经济项目的报道中，把爱奇艺列入“新经济头部企业投资项目”样本。它能说明成都高新区是合理办公判断方向，但不能替代你的实际打卡楼栋。

入职前至少问 HR 或直属同事 5 个问题：

| 问题 | 为什么重要 |
| --- | --- |
| 实际打卡楼栋在哪里？能不能发定位或园区门？ | 成都高新区南部园区、新川、金融城、天府新区不是同一个租房题。 |
| 是固定办公室、项目现场、外包驻场，还是多点协作？ | 驻场或多点协作时，住得太近某一个点反而可能错。 |
| 上下班时间和晚归频率如何？ | 内容 / 审核 / 运营岗位可能更需要看夜间路线、打车点和小区门禁。 |
| 是否有班车或公司推荐通勤方式？ | 有班车时可以外扩，没班车时最后一公里会放大。 |
| 试用期、实习期和转正后办公点会不会变？ | 先短租过渡，比签错一年合同便宜得多。 |

公开报道里的“成都高新区”只能帮助你建立租房方向，不能当成具体楼栋。签租约前，一定要用 HR 发的定位或同事实拍路线，在地图上算小区门到办公楼门的真实时间。
</Callout>

## 第三步：预算和价格口径

成都房租的坑不是“完全租不起”，而是同一区域房源差异很大：新小区、公寓、回迁房、老商品房、合租隔断、商水商电、服务费、短租溢价，都会改变总成本。2026-05-04 这轮整理时，链家 / 贝壳、安居客、58、自如等公开入口存在登录、验证码或动态页面限制，本文不引用单条不可稳定复核的抓取价格。

| 预算段 | 更适合的策略 | 需要接受的代价 |
| --- | --- | --- |
| 2500 元以下 / 月 | 优先合租、老小区、外扩区或短租过渡，先保通勤稳定 | 房龄、室友、隔音、通风和维修风险要现场看清 |
| 2500-3500 元 / 月 | 在天府三街 / 大源外圈 / 新川 / 火车南站之间比较单间或小一居 | 好房源竞争快，假低价和服务费更常见 |
| 3500-5000 元 / 月 | 优先近场一居、公寓或通勤更稳的小区 | 总费用要把物业、水电、网费、服务费一起算 |
| 5000 元以上 / 月 | 可选更近、更稳定或更大面积房源 | 不要只为“离公司近”忽略噪音、采光和合同条款 |

<Callout type="info" title="预算不要只看月租">
把押金、付款周期、中介费、服务费、商水商电、网费、维修责任和提前退租违约金加总后再比较。便宜 300 元但每天多通勤 40 分钟，通常不是好交易。
</Callout>

## 推荐区域详解

### 1. 天府三街 / 天府五街 / 世纪城

这是高新南区互联网 offer 的第一候选圈。优点是接近天府软件园和南部园区办公带，生活配套成熟，晚上回家路线更容易实测；缺点是好房源周转快，房租和中介包装也更硬。

适合：刚入职、实习期短、晚归频率不确定、希望先把通勤风险压低的人。看房时重点查楼道、隔音、厨房排烟、卫生间返味、电梯等待和小区门到地铁口时间。

### 2. 大源 / 锦城湖 / 益州大道

大源适合在高新南区附近办公、又希望生活半径更完整的人。商场、餐饮和小区选择比较多，通勤可以靠骑行、公交、地铁接驳或打车短途解决。

适合：预算中等、希望住得舒服一些、能接受早晚高峰路面波动的人。晚归多的岗位要晚上实测从园区门到小区门的路线，不要只看地图距离。

### 3. 新川 / 广都 / 华府大道

新川和高新区南侧外扩适合预算敏感或办公点更靠南的人。房源性价比可能更好，但通勤是否稳定高度依赖楼栋位置、地铁接驳、共享单车和天气。

适合：能接受 30-45 分钟门到门、希望压低租金、愿意先短租试跑通勤的人。不要在没确认楼栋时直接签新川深处或华阳远端长租。

### 4. 火车南站 / 桐梓林 / 神仙树

这条带适合作为过渡：到高新南区、市中心和金融城都有一定弹性，适合还没确认办公点、双通勤或想先住 1-2 个月校准的人。

适合：入职前信息不完整、预算尚可、希望减少“签错区域”概率的人。缺点是部分房源房龄较老，噪音、停车、楼道、采光和电梯维护要重点查。

### 5. 华阳 / 南湖 / 怡心湖外圈

外扩区适合预算压力大、需要更大面积或有固定班车的人。它不是不能选，而是不适合在楼栋未确认、晚归频率未知时默认首选。

适合：明确办公楼更靠南、通勤路线已经实测、能接受天气和打车波动的人。签约前至少跑一次早高峰和一次晚归路线。

## 城市看房节奏

成都房源选择多，别第一天就签错方向；先看 [成都租房总览](/docs/chengdu) 建立高新南区、新川、金融城、市中心和火车南站的判断框架，再用 1-2 天确认真实楼栋、门到门路线和夜间回家方式。

## 看房核验清单

| 核验项 | 现场要做什么 |
| --- | --- |
| 通勤 | 从小区门到办公楼门完整导航一次，分别看早高峰和晚归路线。 |
| 费用 | 月租、押金、付款周期、中介费、服务费、物业、水电、网费、维修责任逐项写清。 |
| 合同 | 核实房东产权或委托授权，二房东 / 转租必须有书面授权。 |
| 房屋 | 查采光、通风、漏水、墙面霉点、卫生间返味、厨房排烟、空调和热水器。 |
| 安全 | 晚上看路灯、门禁、电梯、楼道、外卖 / 快递动线和打车点。 |
| 合租 | 确认人数、性别、作息、公共区域清洁、访客规则和能否换锁。 |
| 隔断 | 不租违规隔断、无窗房、消防动线差或承诺含糊的房间。 |
| 退租 | 提前退租、转租、押金扣除、维修扣款和违约金要写进合同。 |

## 常见决策模板

| 模板 | 适合情况 | 具体做法 |

| 近场稳妥型 | 入职早期、晚归不确定、预算够 | 先看天府三街 / 天府五街 / 大源，门到门控制在 30 分钟内。 |
