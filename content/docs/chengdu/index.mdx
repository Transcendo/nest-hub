# 成都租房指南

成都租房不要只按“南门”“高新区”粗略判断。互联网、游戏、云计算、企业服务和政企办公锚点主要分在高新南区 / 天府软件园、金融城 / 孵化园、新川、市中心人民南路，以及火车南站换乘带。先确认实际打卡楼栋，再决定住大源、天府三街、天府五街、新川、火车南站还是市中心。

## 怎么使用这个总览

1. 先判断 offer 属于哪条通勤带：高新南区 / 天府软件园、新川、高新区金融城、市中心，还是火车南站换乘带。
2. 再问清 HR 或同事：实际打卡楼栋、园区入口、是否有班车、是否跨园区办公、晚归频率。
3. 看房时用“门到门”时间判断：小区门到地铁口、地铁到园区门、园区门到办公楼都要算。
4. 刚到成都不急着签长租，先用短租、公寓或同事样本校准一周通勤，再决定住哪一圈。

## 地图判断原则

成都租房先看办公区，再看预算：

- **高新南区 / 天府软件园**：成都互联网和科技 offer 的核心判断区，重点比较天府三街、天府五街、世纪城、大源、锦城湖、益州大道和南湖外扩。
- **新川 / 高新区南侧外扩**：适合预算敏感或办公点在新川、AI 创新中心、新经济项目南侧的人，必须确认地铁接驳和夜间回家方式。
- **金融城 / 孵化园**：通勤和生活都比较均衡，但租金更硬，适合预算更高或需要兼顾市中心生活的人。
- **人民南路 / 市中心**：市中心政企、职能、销售和企业业务办公点优先看地铁换乘、房龄、噪音和老小区维护。
- **火车南站 / 桐梓林 / 神仙树**：适合刚入职过渡、双通勤或还没确认办公点的人，但要接受换乘和租金溢价。

## 成都找房方式

- **不要只听“公司在高新区”**：高新区南部园区、金融城、新川、天府新区不是同一个租房题，楼栋差几公里就会换一套策略。
- **先实测门到门**：成都路面和园区尺度会放大最后一公里，地图上的地铁近不等于下班后回家轻松。
- **预算和生活方式一起看**：大源、天府三街更省通勤但租金和房源竞争更强；新川和南湖外扩更便宜但通勤稳定性要验证。
- **签前拆清费用**：中介费、服务费、押付、民水民电、维修、提前退租和转租授权都要落到合同或聊天记录里。
