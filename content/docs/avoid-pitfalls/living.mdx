# 入住生活

> 成功入住只是开始，舒服的租房生活靠的是持续的细节管理。

## 入住记录验收

### 必做记录

- 全屋拍照或录像
- 水、电、燃气表读数拍照
- 家具家电状态记录
- 发现的问题及时书面确认

### 入住验收清单

- 房屋整体清洁和损坏情况
- 门窗、锁具和钥匙数量
- 水电设施和热水器
- 厨房设备和卫生间设施
- 空调、洗衣机、冰箱等家电
- 网络账号密码和物业联系方式

## 合租相处指南

### 入住前先谈清楚

- 作息时间
- 卫生标准与分工
- 公共费用分摊
- 访客和留宿规则

### 日常原则

- 深夜和清晨控制噪音
- 公共区域用完就收拾
- 有问题及时说，不要靠憋着
- 贵重物品自己保管

## 日常维护责任

### 租客责任

- 日常清洁和保养
- 小件消耗品更换
- 轻微堵塞和基础整理

### 房东责任

- 房屋结构问题
- 基础设施故障
- 大型家电非人为损坏

### 报修流程

1. 拍照记录问题。
2. 电话加文字同步给房东。
3. 明确响应时间和责任方。
4. 维修后再确认是否解决。

## 常见生活问题

- 邻里噪音：先友好沟通，再找物业协调。
- 室友矛盾：围绕规则和事实谈，不要把问题情绪化。
- 房东关系：按时交费，问题及时同步。

## 本章要点

1. 入住当天的记录，直接决定退租时你能不能自证。
2. 合租规则越早说清楚，后面越省事。
3. 报修一定留痕，不要只打电话。
4. 生活品质改善可以做，但别越过合同边界。
