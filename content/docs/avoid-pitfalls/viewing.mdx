# 实地看房

> 百闻不如一见，实地看房是租房决策的关键环节。

## 看房时机

### 最佳看房时间

- 工作日晚上：看噪音、灯光和生活节奏
- 周末白天：看采光、小区环境和人流
- 雨天或雨后：看漏水和排水
- 夜间：看治安和回家动线

### 看房次数建议

- 初次看房：整体筛选
- 二次看房：验证细节
- 夜间复看：确认噪音与安全感

## 房屋检查清单

### 结构与安全

- 墙体是否有裂痕和渗水
- 门窗开关是否顺畅
- 天花板是否有漏水痕迹
- 地板是否平整
- 阳台和护栏是否安全

### 水电设施

- 水压是否稳定
- 插座数量和位置是否够用
- 照明是否正常
- 网络信号是否稳定
- 开关插座是否有松动

### 生活设施

- 空调冷热是否正常
- 热水器是否稳定
- 油烟机和燃气灶是否好用
- 马桶、地漏和下水是否顺畅
- 洗衣机、冰箱等家电状态是否正常

### 环境细节

- 采光和通风是否达标
- 是否有霉味或新装修刺激气味
- 隔音是否在可接受范围
- 收纳空间是否够用

## 小区环境评估

### 安全性

- 门禁是否有效
- 监控覆盖是否完整
- 是否有保安和值守
- 夜间照明是否充足

### 周边配套

- 快递代收是否方便
- 垃圾清运是否及时
- 电梯和停车位是否够用
- 超市、餐饮和交通站点是否就近

## 看房记录建议

### 拍照内容

- 客厅、卧室、厨房、卫生间
- 水表、电表、燃气表
- 家具家电细节
- 有问题的位置

## 常见陷阱

- 照片与实物不符
- 用样板间替代真实出租房
- 故意隐瞒噪音、邻里纠纷或设施故障
- 现场持续催促，逼你立刻交钱

## 本章要点

1. 白天和晚上最好都看一次。
2. 拍照和做笔记不是形式，是后续谈判和维权材料。
3. 房屋内部和小区外部要一起看。
4. 看完不要急着决定，至少给自己一个复核窗口。
