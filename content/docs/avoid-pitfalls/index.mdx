# 避坑指南

> 由巢聚-旦聚旦团队整理，给第一次独立租房的人一套能直接落地执行的流程。

## 指南概述

这套内容按真实租房流程组织，不是零散建议堆砌。建议先建立统一标准，再去看不同城市和不同办公区的具体页面。

## 章节导览

### [租前准备](./preparation)

- 预算规划和费用预估
- 区域选择和通勤考量
- 找房渠道和信息筛选
- 识别虚假房源信号

### [实地看房](./viewing)

- 最佳看房时间安排
- 房屋质量检查清单
- 小区环境评估要点
- 看房记录和对比技巧

### [签约谈判](./contract)

- 谈判策略和技巧
- 合同条款审核要点
- 识别合同陷阱
- 押金和费用保障

### [入住生活](./living)

- 入住记录和验收
- 合租相处指南
- 日常维护责任
- 问题处理流程

### [常见陷阱](./traps)

- 中介常见套路
- 费用陷阱识别
- 房屋质量陷阱
- 紧急情况应对

### [维权指南](./rights)

- 投诉举报渠道
- 证据收集要点
- 法律维权途径
- 纠纷预防建议

## 建议阅读顺序

1. [租前准备](./preparation)
2. [实地看房](./viewing)
3. [签约谈判](./contract)
4. [常见陷阱](./traps)
5. [维权指南](./rights)

## 使用建议

- 先把自己的预算、通勤底线和合同底线写下来，再开始看房。
- 看房和签约时把页面里的检查项当作现场清单使用。
- 读完后再进入 [北京](/docs/beijing)、[上海](/docs/shanghai)、[杭州](/docs/hangzhou)、[深圳](/docs/shenzhen) 城市页。
