# 避坑指南

<Callout type="success" title="先看结论">
租房不要从“哪套看起来不错”开始，而要先排除不该租的房子。先写清预算、通勤、噪音、楼层、合租和合同底线，再用白天 / 夜晚两次看房、全屋拍照留证、合同逐项落字来降低风险。
</Callout>

## 使用建议

- 先把自己的预算、通勤底线和合同底线写下来，再开始看房。
- 看房和签约时把页面里的检查项当作现场清单使用。
- 涉及合同和法律判断时，以合同原文、法律法规和专业意见为准。
- 读完后再进入 [北京](/docs/beijing)、[上海](/docs/shanghai)、[杭州](/docs/hangzhou)、[深圳](/docs/shenzhen) 城市页。
