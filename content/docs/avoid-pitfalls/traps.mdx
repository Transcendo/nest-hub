# 常见陷阱

> 知道别人最常怎么坑你，才能在现场第一时间识别出来。

## 中介陷阱

### 虚假房源

- 套路：用低价房吸引你到现场，再推高价房。
- 识别：价格明显低于市场价。
- 应对：提前查行情，坚持只看目标房源。

### 隐性收费

- 套路：押一付一只是表面，后面再加服务费、清洁费、管理费。
- 识别：费用口径前后不一致。
- 应对：所有费用要求一次性书面列清。

### 假房东

- 套路：冒充房东先收押金或定金。
- 识别：身份和产权文件拿不出来。
- 应对：核对身份证、房产证、授权文件。

## 费用陷阱

- 商业水电按民用标准宣传
- 维修责任全部推给租客
- 退租时临时加收清洁费、翻新费

## 房屋陷阱

- 隔断间：消防和隔音问题都很重
- 甲醛房：刚装修完就出租
- 历史纠纷房：噪音、邻里矛盾、治安问题被刻意隐瞒

## 防范动作

1. 先查市场价，再谈房源。
2. 只相信书面条款，不相信嘴上承诺。
3. 拍照、录视频、留聊天记录。
4. 不被“今天不定就没了”这类话术推着走。

## 本章要点

1. 便宜得不合理，大概率就不真实。
2. 费用问题不问清，后面最容易扯皮。
3. 房屋安全问题比装修好看更重要。
4. 冷静和留痕是最有效的反套路动作。
