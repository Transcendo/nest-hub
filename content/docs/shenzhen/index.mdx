# 深圳租房指南

深圳不是一个“住南山就行”的城市。科技公司分布在南山科技园 / 深圳湾、西丽 / 留仙洞、坂田、福田 CBD、前海、坪山、盐田、宝安北部等完全不同的通勤系统里。

这页先作为深圳地区互联网、硬科技、金融科技和先进制造公司的总览入口，建立官方来源支撑的公司样本池与通勤带判断。公司级租房指南后续再按真实办公楼栋逐页拆分。

## 当前口径

- 只使用公司官网、投资者关系页、政府公开信息等官方可信来源。
- 不使用来源不明的网站、灰色接口、非必要第三方抓取数据。
- 地图坐标是通勤方向示意，不替代实时地图路线。
- 当前只做深圳公司样本池和办公区通勤判断，不编造未经核验的房源、价格和中介渠道。
