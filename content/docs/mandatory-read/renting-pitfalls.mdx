# 租房避坑指南

<callout emoji="⚠️" background-color="light-yellow">
这份清单不是给中介看的，是给自己保命用的。
</callout>

## 先看什么

### 1. 位置和周边

- 避开小学、游乐设施、临街、施工点
- 女生尽量避开要穿巷子的房子
- 一楼潮湿、吵、采光差，顶楼容易热和水压问题
- 白天看采光，晚上看噪音和路灯

### 2. 房屋质量

- 刚装修完、首次出租的房子先谨慎
- 检查甲醛、渗水、墙体隔断、下水、花洒、水压、马桶
- 敲墙判断是不是隔断间
- 检查空调、插座、门锁、窗户

### 3. 合同和费用

- 问清押一付一还是押二付一
- 问清起租时间、涨租、宽带、物业、水电气网
- 定金和订金别混
- 所有口头承诺都写进合同
- 退租、转租、押金退还时间必须写清

### 4. 合租

- 先了解室友作息、卫生、宠物、带人、性别和关系结构
- 尽量不要三人以上复杂合租
- 公区卫生和厨房情况，基本决定合租质量

## 看房时的硬动作

1. 白天看一次，晚上看一次
2. 每个房间都拍照留底
3. 水电燃气初始读数记录下来
4. 家电家具逐项确认
5. 合同先看再交钱

## 一句话原则

> 能写进合同的才算数，别信嘴。
