# 广州租房指南

广州租房不能只看“离珠江新城远不远”。腾讯 / 微信相关团队看海珠与琶洲，网易游戏和部分互联网团队看天河 / 科韵路 / 智慧城，自动驾驶、硬件和医疗科技看黄埔科学城，汽车、游戏、电商和产业总部还会落到番禺、南沙。

<GuangzhouOverview />

## 怎么使用这个总览

1. 先判断 offer 属于哪条通勤带：琶洲 / 海珠、天河 / 科韵路、黄埔科学城、番禺、珠江新城，还是南沙。
2. 再进入公司页，核实际办公楼栋、园区入口、候选居住区、通勤路线和签约风险。
3. 看房前问清 HR 或同事：是否跨园区、是否有班车、常见下班时间、是否需要驻场或频繁跨城。
4. 看房当天用正规平台、线下实看、地铁 / 骑行 / 打车实测和合同条款交叉确认。

## 地图判断原则

广州要按办公区拆开判断：

- **琶洲 / 海珠**：腾讯 / 微信、部分内容和平台团队更看重 8 / 18 号线、琶洲西区、赤岗、客村、磨碟沙和万胜围。
- **天河 / 科韵路 / 智慧城**：网易游戏、游戏与互联网团队要区分科韵路、车陂、员村、棠下和天河智慧城。
- **黄埔科学城**：自动驾驶、硬件和医疗科技岗位要先确认园区入口、班车、地铁接驳和夜间打车。
- **番禺 / 大学城 / 万博**：游戏、电商、汽车相关岗位要比较番禺广场、万博、大学城、南村和广州南站外扩。
- **珠江新城 / 越秀 / 荔湾**：适合金融、职能和跨区通勤人群，但租金、楼龄和换乘成本要一起算。
- **南沙**：产业基地和汽车相关岗位必须先确认班车、自驾、地铁末端和晚归路线。

## 广州找房方式

- **先问楼栋，不先看房**：广州办公点跨度很大，同一家公司不同团队可能对应完全不同的居住圈。
- **通勤要分昼夜测**：白天看地铁和骑行，晚上看公交收班、打车点、小区门禁和路灯。
- **房源形态要看清**：老小区、城中村、公寓、产业园周边房源混在一起，图片漂亮不代表合同安全。
- **费用拆到明细**：中介费、服务费、押付、商水商电、网络费、维修责任、转租授权和退租违约金都要确认。

## 资料来源与口径

广州总览优先使用公司官网、投资者关系页、政府 / 园区公开信息确认办公锚点；租房建议只保留能帮助判断通勤、片区和看房风险的信息。实时租金和房源状态变化快，页面不写单次抓取价格，具体房源仍以正规平台、线下实看和合同为准。
