# 广州租房指南

## 使用顺序

1. 先在总览里判断你的 offer 属于琶洲 / 海珠、天河、黄埔、番禺、珠江新城还是南沙通勤题。
2. 进入公司页后，先照着“办公室核验”问 HR 或同事，不要先看房。
3. 只在楼栋、班车、打卡门和常见下班时间清楚后，再把候选小区分成近场、均衡、外扩三类。
4. 看房当天用正规平台和线下实看交叉核验价格，本项目不把第三方截图当稳定事实。

## 资料来源与口径

广州页优先使用公司官网、投资者关系页、政府/公共交通公开信息。公司页里的区域建议是 NestHub 面向年轻租房者的编辑判断，不代表公司官方建议，也不构成房源推荐。
