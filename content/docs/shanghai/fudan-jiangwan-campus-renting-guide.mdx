# 复旦大学江湾校区研究生租房指南

<Callout type="success" title="先看结论">
复旦江湾校区研究生租房，第一优先级不是“离校最近”，而是 **你每天主要在哪个校区 / 实验室 / 教学楼活动**。如果核心活动在江湾，优先看 **新江湾城、国帆路、国权北路、三门路、殷行路**；如果经常去邯郸校区或五角场，优先看 **五角场、政立路、国定路、江湾体育场**；如果预算更紧，再往 **中原、市光路、嫩江路、高境、江湾镇** 扩。

研究生最常见的可行区间：**合租单间 1700-3200 元/月；老小区一居 3000-4500 元/月；品牌公寓单间 / 一居 2500-6500 元/月；两居整租 5500-9000 元/月；新江湾城品质小区三居常见 10000 元/月以上**。
</Callout>

## 这篇指南适合谁

适合这些场景：

- 你是复旦江湾校区研究生、博士生、联合培养学生、访问学生或刚来江湾做科研的人；
- 你不确定该住校内、校门口、新江湾城、五角场，还是往三门路 / 中原方向找；
- 你在链家 / 贝壳、58、房天下、自如、豆瓣、小红书、微信群之间来回看，信息太散；
- 你关心的不只是租金，还包括 **通勤、做饭、安静、短租、室友、实验室晚归、安全和合同风险**。

<Callout type="info" title="校区位置口径">
复旦大学江湾校区官方信息显示，江湾校区位于 **上海市杨浦区淞沪路 2005 号**，距邯郸校区约 3.5 公里，校区师生超过 1 万人。对租房来说，江湾不是单点，而是一整个 **新江湾城 - 五角场 - 三门路 - 中原** 的居住带。
</Callout>

## 评价信息：从学生视角怎么判断

### 公寓

优点：

- 拎包入住，适合刚来上海、短租、实习、访问学生；
- 管理相对标准化，女生独居安全感更强；
- 部分支持 1-3 个月短租或押一付一。

缺点：

- 单平米价格通常高于普通住宅；
- 面积小，隔音和采光差异很大；
- 服务费、水电、退租扣款规则可能让总成本上升。

适合：第一学期过渡、没时间看房、女生独居、预算 3500 元以上。

### 小区合租

优点：

- 总价最可控，1700-3200 元/月能看到不少线索；
- 如果室友靠谱，生活成本低；
- 同学 / 校友转租有机会拿到更真实的房源。

缺点：

- 室友作息、卫生、访客、宠物都是变量；
- 二房东、隔断、转租授权风险更高；
- 好房源流转很快。

适合：预算敏感、能接受合租、愿意花时间筛选的人。

### 老小区独居

优点：

- 比公寓更有生活感，可能能做饭；
- 一居 3000-4500 元/月在三门路、中原方向能看到；
- 私密性比合租强。

缺点：

- 楼龄、卫生间、厨房、下水、采光、电梯要仔细看；
- 装修和家具可能比较老；
- 女生独居要更看重门禁、楼道和夜间路线。

适合：博士长期住、想安静写论文、不想合租的人。

## 到沪看房执行方案

### 到上海前 48 小时

- 确认你主要活动点：江湾哪栋楼、邯郸是否高频、晚归频率；
- 明确预算：月租、押金、中介费、服务费、通勤费；
- 线上保留 8-12 套候选，不要只收藏截图；
- 做表格：小区 / 公寓、价格、户型、面积、付款方式、联系人、合同主体、可入住时间。

### 第 1 天：跑近校圈

重点看 **新江湾城、国帆路、国权北路、三门路**。

目标是建立“离校近到底贵多少”的价格感：能不能接受小房间、是否愿意为近校多付 500-1500 元/月。

### 第 2 天：跑五角场和预算圈

重点看 **五角场、政立路、国定路、中原、殷行路**。

这一天比较的是：多通勤 10-20 分钟，能不能换来更低总价、更大面积、更方便吃饭或更好的室友。

### 第 3 天：复看和谈合同

只复看最满意的 2-3 套，最好晚上复看一次。

重点看：楼道灯、门禁、电梯、夜间回家路线、临街噪音、室友状态、卫生间反味、厨房和洗衣晾晒。

## 看房核验清单

### 房子本身

- 采光：不要只看中介白天广角图；
- 噪音：晚上复看，尤其五角场、政立路、主路边；
- 卫生间：查反味、漏水、地漏、热水器、水压；
- 厨房：确认能不能做饭、燃气还是电磁炉；
- 空调和取暖：上海冬天湿冷，空调制热比想象重要；
- 洗衣晾晒：研究生长期住，晾衣空间很实际；
- 电梯和门禁：女生独居、晚归尤其要看。

### 合同和钱

- 房东身份、房本或授权链路是否清楚；
- 转租是否有原房东书面同意；
- 押金退还条件是否写进合同；
- 中介费、服务费、保洁费、维修费是否一次讲清；
- 水电燃气民用还是商用；
- 能否办理居住登记；
- 是否隔断、群租、不可备案。

### 学生特殊项

- 能否短租 3-6 个月；
- 寒暑假是否空置，能否转租；
- 是否允许自行更换门锁 / 加装安全设施；
- 室友是否同校 / 同性 / 作息相近；
- 从实验室晚归是否有稳定路线；
- 是否方便去邯郸校区、五角场、图书馆、食堂和医院。

## 常见决策模板

| 你的情况 | 推荐优先级 | 不建议 |
| --- | --- | --- |
| 江湾实验室高频，预算 2500-3500 | 三门路 / 殷行路合租、自如寓小房型、校友转租 | 为了近校硬租高价新小区单间 |
| 女生独居，预算 3500-5000 | 冠寓、城投宽庭、COZI、小一居、可信同学转租 | 远程定金、无门禁老楼、夜路复杂区域 |
| 博士长期住，预算 4000-6000 | 三门路 / 中原一居，或两人合租两居 | 只看装修图，不看噪音和下水 |
| 经常去邯郸和五角场 | 五角场、政立路、国定路、江湾体育场 | 住到过北或过远导致两边都不方便 |
| 预算极限 1700-2500 | 中原、殷行路、财大附近合租，校友群转租 | 低价房没看合同就付款 |
| 情侣 / 两人合租 | 三门路 358/510、时代花园、三湘未来海岸、同济欣苑 | 一味追新江湾城品质盘导致总价失控 |
