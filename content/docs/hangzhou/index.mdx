# 杭州租房指南

杭州互联网公司不是均匀摊在一个市中心，而是明显分成几条通勤带：城西未来科技城 / 西溪、滨江高新区、西湖黄龙 / 西溪路、拱墅大运河、萧山信息港。

这页作为杭州地区互联网大厂与数字科技公司的总览入口，并已按公司拆出独立租房指南，方便按办公区和通勤带逐页判断。

## 当前口径

- 只使用公司官网、投资者关系页、政府公开信息等官方可信来源。
- 不使用来源不明的网站、灰色接口、非必要第三方抓取数据。
- 地图坐标是通勤方向示意，不替代实时地图路线。
- 杭州公司指南已优先覆盖首批 20 家互联网与科技公司，后续继续按真实办公点、通勤反馈和租赁市场变化校准。
